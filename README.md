## Installation

### Command line using composer

```bash
composer require bulldog-creative/slimcd
```

### composer.json

```json
{
    "require": {
        "bulldog-creative/slimcd": "2.0.*"
    }
}
```

* [v2.0.0 zip](https://github.com/BulldogCreative/SlimCD/archive/v2.0.0.zip)
